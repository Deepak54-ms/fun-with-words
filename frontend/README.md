# Fun with Words

A small bilingual vocabulary app for children built with React, TypeScript, and Vite.

## Features

- English/Hindi language toggle powered by `react-i18next`
- Colorful image cards with simple sentences
- Pronunciation using the Web Speech API
- Quiz activity to match words with pictures

## Development

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Build the project for production:

```bash
npm run build
```
